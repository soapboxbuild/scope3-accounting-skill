description: Calculate PCAF financed emissions for a real estate portfolio (Cat 15)
argument-hint: [portfolio with debt/equity exposure and property values]
---
Calculate PCAF financed emissions (Category 15) for this portfolio:

For each asset:
1. Attribution factor = outstanding loan or equity / current property value
2. Asset emissions = Scope 1 + 2 + 3 (use available data or property-type proxy)
3. Financed emissions = attribution factor × asset emissions
4. PCAF data quality score per asset
5. Portfolio totals: absolute financed emissions + emissions intensity (tCO2e/$M invested)
6. Compare to CRREM pathway to identify misaligned assets

Present per PCAF standard format with data quality disclosure.
